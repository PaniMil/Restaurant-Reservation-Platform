This project is a full-stack restaurant reservation platform developed to simplify the process of discovering restaurants, making reservations, and managing restaurant information.

Customers can browse restaurants, search by name, filter restaurants by category, view restaurant details, reserve tables, manage their reservations, rate restaurants after completed reservations, and save their favorite restaurants.

The system also includes a dedicated Admin Dashboard that allows administrators to manage restaurants, users, reservations, and customer reviews through a centralized interface.

-Features
--Customer
    User registration and login
    Profile management
    Change password
    Browse restaurants
    Search restaurants by name
    Filter restaurants by category
    View restaurant details
    Table reservation
    Reservation management
    Cancel reservations
    Rate and review restaurants
    Favorite restaurants
--Admin
    Dashboard overview
    Restaurant management (CRUD)
    User management
    Reservation management
    Review management
    Technologies

    React Router
    Tailwind CSS
-Backend

    Express.js
-Database
